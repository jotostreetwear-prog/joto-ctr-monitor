# Скилы для продаж на маркетплейсах

Набор «деньги + разведка» для Claude Code: финансовый блок и анализ конкурентов
для продавца на маркетплейсах (Wildberries, Ozon, Яндекс Маркет).

## Что внутри

Финансы (Small Business):
- `margin-analyzer` — юнит-экономика и маржа по SKU с учётом всех комиссий
- `price-check` — аудит своих цен, поиск убыточных позиций, проверка перед акциями
- `cash-flow-snapshot` — движение денег и кассовый разрыв
- `close-month` — закрытие месяца: чек-лист, сверка, P&L
- `tax-prep` — структурирование данных для налогов (не консультация)

Разведка (Bright Data):
- `price-comparison` — сравнение цен с конкурентами
- `competitive-intel` — профиль конкурента: ассортимент, цены, оценка продаж
- `brand-listening` — анализ отзывов, тональности и спроса

Скилы ссылаются на встроенный скил `xlsx` для выгрузок в Excel — он уже доступен.
Для реальных данных по конкурентам/ценам/отзывам подключи коннектор сбора
данных (Bright Data / Nimble / маркетплейсный MCP) или используй веб-поиск.

## Установка

Скопируй папки в `.claude/skills/` своего проекта (доступны всем в репозитории):

```bash
cp -r skills/* .claude/skills/
```

Или для себя во всех проектах — в `~/.claude/skills/`:

```bash
cp -r skills/* ~/.claude/skills/
```

Затем в Claude Code выполни `/reload-plugins` (или перезапусти сессию).
Проверить, что подхватились: спроси что-нибудь вроде
«посчитай маржу по этому товару» — должен сработать `margin-analyzer`.

## Структура

```
skills/
├── margin-analyzer/SKILL.md
├── price-check/SKILL.md
├── cash-flow-snapshot/SKILL.md
├── close-month/SKILL.md
├── tax-prep/SKILL.md
├── price-comparison/SKILL.md
├── competitive-intel/SKILL.md
└── brand-listening/SKILL.md
```

Каждый SKILL.md — это инструкция в формате с YAML-шапкой (`name`, `description`)
и телом. `description` отвечает за то, когда скил срабатывает; тело — за то, как
именно Claude выполняет задачу. Правь под свои площадки и процессы.
