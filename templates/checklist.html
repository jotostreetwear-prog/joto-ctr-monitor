Готовность карточек по артикулам

ℹ️ Как пользоваться чек-листом
  • 🟩 зелёный квадрат — пункт выполнен / характеристика заполнена; 🟥 красный — нет.
  • Слева — метрики готовности (фото, видео, баркод, рейтинг, СЕО и т.д.), справа — характеристики из карточки WB (Состав, Вид застёжки, Сезон…). Все характеристики тянутся из кабинета WB.
  • ✋ Колонки с пометкой «вручную» и все характеристики можно переключать кликом по квадрату (красный ↔ зелёный). Отметка сохраняется навсегда — переживает обновление и не сбрасывается.
  • Балл (справа) — итоговый % готовности по метрикам.
  • Тумблер «Только с недочётами» — показать карточки, где есть что доделать.
  • Тумблер «Только красные» — спрятать зелёные квадраты, видно только незаполненное.
  • Список категорий — показать одну категорию (Шорты, Худи…); характеристики подстроятся под неё.
  • Клик по ▸ у артикула — раскрыть все характеристики со значениями; клик по названию — открыть карточку на Wildberries.
  • Данные обновляются автоматически каждый час; кнопка «Обновить» — пересчитать прямо сейчас.

✋ Эти колонки отмечаешь вручную (кликом по клетке, отметка сохраняется):
• Закреп. отзывы, Блокировка акций — по умолчанию красные, кликаешь те, где сделано → зелёные.
• Сертификаты — по умолчанию зелёные, кликаешь те, где серта нет → красные.
Остальные метрики (включая рекомендации и размерную сетку) считаются автоматически из данных Wildberries.
